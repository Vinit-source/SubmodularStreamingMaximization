class Matrix¶

This is a simple Matrix class for quadratic \( N \times N \) matrices. The Matrix is implemented with a 1d (column major) std::vector. There are also some linear algebra functions available.

Public Functions

inline Matrix(Matrix const &other, unsigned int N_sub)¶

Copies the upper left N_sub x N_sub matrix from other into the new object. Caller has to make sure that N_sub <= other.size()

Parameters

  • &other – The matrix from which we want to copy entries

  • N_sub – The size of the sub matrix. Caller has to make sure that N_sub <= other.size()

Returns A

newly constructed N_sub x N_sub Matrix object.

inline void replace_row(unsigned int row, data_t const *const x)¶

Replaces the row at position row with the given vector. Caller has to make sure that row < N and that x has at-least N elements.

Note

There are no safety checks performed.

Parameters

  • row – The row which should be replaced.

  • x – The vector which the row should be replaced with

inline void rank_one_update(unsigned int j, data_t const *const x)¶
