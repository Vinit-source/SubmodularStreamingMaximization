inline Matrix cholesky(Matrix const &in, unsigned int N_sub)¶

Computes the choleksy decomposition of the N_sub x N_sub sub matrix and returns the lower triangular matrix L with LL^T = in.

Parameters

  • &in – The matrix which should be decomposed.

  • N_sub – The N_sub x N_sub sub-matrix. The caller has to make sure that N_sub <= N. If you want to print the entire matrix supply N_sub = N.

Returns Returns

the cholesky decomposition
